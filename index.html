Course Work

Potato Sweeper

This is a web-based minesweeper clone with potatos as mines written in C++ and HTML using CGI. On a 10x10 grid, the player can choose either to flag the potatoes or to dig a hole. The game ends if the user flags all 10 potatoes or if they dig up a potato. The user can use up to 10 flags.

SpaghettiOS: Covid-19 Trading

This is a group project that was a created in response to Covid-19 that were affecting local communities writting in C++ and HTML. It is a trading platform where each user can create a unique account, enter in items they want, and then matched with potential traders who have the item.
